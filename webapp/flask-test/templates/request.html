How To Manage JSON Requests

This is the JSON data, as a string:
{{jsondata}}
This is the JSON data, as an array of objects:
    {% for item in data %}
  • {{item.id}}: {{item.type}}
    • {% endfor %}

Code to manage a JSON Request

@app.route('/request', methods=['POST'])
def jsonreq():
    # Get the JSON data sent from the form
    jsondata = request.form['jsondata']
    # Convert the JSON data into a Python structure
    data = json.loads(jsondata)
    return render_template('request.html', data=data, jsondata=jsondata)
